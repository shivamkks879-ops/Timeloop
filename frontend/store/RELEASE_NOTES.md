# Release Notes — Time Loop Escape

---

## v1.0.0 — Launch (June 2026)

🎉  Welcome to the Time Lab! First public release.

### Content
- **100 hand-crafted levels** across 8 worlds.
- **World 8 boss fight** — CHRONOS.
- 6 unlockable robot skins gated by progression.
- 10 achievements — from *First Steps* to *Chronos Slain*.
- Statistics dashboard: playtime, deaths, echoes, fastest clear.

### Feel & polish
- Custom Skia-rendered vector character with 8 poses.
- Animated parallax star field, particle bursts on jump/land/portal/key/death/rewind.
- Screen shake on death and rewind (toggleable).
- Original synth soundtrack + tactile haptics per action.

### Controls
- Zero input delay via ref-based touch tracking.
- Multi-touch: LEFT + JUMP or RIGHT + JUMP simultaneously.
- One-thumb mode (compact bottom-left cluster).
- Configurable button opacity (40 – 100%).

### Accessibility
- Colorblind-safe palette flag.
- Independent music and SFX toggles.
- Screen-shake toggle.
- Fully landscape-locked, immersive fullscreen edge-to-edge.

### Under the hood
- Deterministic 60 TPS simulation — every solution is reproducible.
- Headless auto-playtest verifies all 100 levels clear with S-grade solutions on every build.

Enjoy the loop.
